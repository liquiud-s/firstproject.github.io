**TaskFlow – Student Deadline Management Prototype**

TaskFlow is a simple web prototype designed to help university students manage academic deadlines more effectively. The project was developed as part of a Human–Computer Interaction (HCI) design process and focuses on solving problems caused by fragmented communication tools such as WhatsApp, LMS platforms, email, and classroom announcements. Through interviews with undergraduate students, the research identified that many students struggle to track assignments because important information is spread across multiple platforms. As a result, deadlines are often discovered too late, leading to rushed work and unnecessary stress. TaskFlow addresses this problem by providing a centralized interface where students can clearly view upcoming tasks and deadlines. The design focuses on simplicity, visibility, and task prioritization to reduce cognitive load and help students stay organized.

**Features**
1. Welcome screen introducing the application
2. Task dashboard displaying upcoming deadlines
3. Task detail screen showing assignment information
4. Ability to mark tasks as completed
5. Clear visual indicators for urgent tasks
